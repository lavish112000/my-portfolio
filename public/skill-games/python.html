2. Labeling Ingredients (Variables)

In cooking, you label your ingredients. In Python, you use variables to store information and give it a name. You use the equals sign (=) to assign a value to a variable.

Try labeling your own ingredient below!

3. Types of Ingredients (Data Types)

Recipes use different kinds of ingredients: flour (text), sugar amount (numbers), a list of spices. Python also has different data types:

  • Strings: Text, like "Hello" (in quotes)
  • Numbers: Integers (10) or decimals (3.14)
  • Lists: Ordered collections, like ["salt", "pepper"]

4. Checking the Oven (If/Else)

Recipes often have checks: "If the oven is preheated, put the cake in, else wait." Python uses if, elif (else if), and else statements to make decisions based on whether a condition is true or false.

temperature = 200
if temperature >= 180:
  print("Bake!")
else:
  print("Preheat...")

Oven Status

5. Chopping Vegetables (Loops)

Sometimes you need to repeat a step, like chopping multiple carrots. Python uses loops (like for loops) to execute a block of code multiple times, often iterating over items in a list.

vegetables = ["🥕", "🧅", "🥔"]
for veg in vegetables:
  print("Chopping", veg)
🥕

6. Reusable Recipe Steps (Functions)

Recipes often have common steps like "mix dry ingredients." In Python, you can group reusable code into functions using the def keyword. You give the function a name and can then "call" it whenever you need to perform that task.

def mix_ingredients(ing1, ing2):
  print("Mixing", ing1, "and", ing2)

mix_ingredients("Flour", "Sugar")
