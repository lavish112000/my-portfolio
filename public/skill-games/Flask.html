What is Flask?

Flask is a lightweight web framework for Python. It's designed to make getting started quick and easy, with the ability to scale up to complex applications.

Think of Flask as the foundation and structure of a building - it provides the basic structure, and you add what you need on top of it.

How Flask Works

Flask follows a simple request-response cycle:

1. User sends request → 2. Flask processes request → 3. Flask sends response

When a user visits a URL, Flask matches the URL to a Python function, which processes the request and returns a response.

Basic Flask Code

Here's a simple Flask application:

from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, World!"

if __name__ == '__main__':
    app.run(debug=True)

This code creates a basic web server that responds with "Hello, World!" when someone visits the homepage.

Real-World Scenario: Online Bookstore

Imagine you're building a website for an online bookstore. Here's how Flask would help:

  • Homepage: Visitors see book recommendations
  • Search: Users can search for specific books
  • Book Details: Each book has its own page
  • Shopping Cart: Users can add books to their cart

Flask makes it easy to create each of these pages and connect them to your database.
