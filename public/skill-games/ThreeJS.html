Three.js Visualizer

An interactive guide to the fundamentals of 3D graphics on the web. Explore each core concept and see how it contributes to the final scene.

1. The Scene

The Scene is the most fundamental component. Think of it as a stage or a universe where all your 3D objects, lights, and cameras will live. It's the container for everything that appears in your 3D world.

// Create a new scene object const scene = new THREE.Scene(); // Set a background color scene.background = new THREE.Color(0x1a0442);

2. The Camera

The Camera is your eye into the 3D world. It determines what you see and from what perspective. The most common type is the PerspectiveCamera, which mimics how the human eye sees—objects farther away appear smaller.

// Create a perspective camera // (FOV, Aspect Ratio, Near Clip, Far Clip) const camera = new THREE.PerspectiveCamera(75, width / height, 0.1, 1000); // Position the camera back from the origin camera.position.z = 5;

3. Geometry & Material

To create a visible object (a Mesh), you need two things: a Geometry which defines its shape, and a Material which defines its surface appearance (color, texture, shininess).

// Define the shape (a box) const geometry = new THREE.BoxGeometry(); // Define the surface (a standard white material) const material = new THREE.MeshStandardMaterial({ color: 0xffffff }); // Combine them into a Mesh const cube = new THREE.Mesh(geometry, material); // Add the mesh to the scene scene.add(cube);

4. Lights

Most materials won't be visible without a Light source. There are many types of lights. AmbientLight provides a soft, general illumination, while DirectionalLight acts like a distant source, like the sun, casting shadows.

// Add a soft ambient light const ambientLight = new THREE.AmbientLight(0x404040); scene.add(ambientLight); // Add a directional light from an angle const directionalLight = new THREE.DirectionalLight(0xffffff, 0.8); directionalLight.position.set(1, 2, 3); scene.add(directionalLight);

5. Renderer & Animation Loop

The Renderer takes the scene and camera and draws the result onto an HTML <canvas>. To create motion, we use an Animation Loop—a function that repeatedly calls the renderer, updating the object's position or rotation each time to create smooth animation.

// Create a renderer and add it to the page const renderer = new THREE.WebGLRenderer(); document.body.appendChild(renderer.domElement); // The animation loop function animate() { requestAnimationFrame(animate); cube.rotation.y += 0.01; // Rotate the cube renderer.render(scene, camera); // Render the scene } animate();
